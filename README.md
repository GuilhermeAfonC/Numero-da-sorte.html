# 🎲 Número Surpresa

Uma aplicação web interativa que simula um "oráculo digital", permitindo ao usuário fazer perguntas e receber respostas aleatórias geradas pelo sistema.

---

## 🚀 Funcionalidades

* 🎲 Geração aleatória de respostas
* 💬 Interface simples e intuitiva
* 📜 Histórico de perguntas realizadas
* ⚡ Comunicação assíncrona com API (Fetch API)
* 🛡️ Validação de dados no front-end e back-end
* ❗ Tratamento de erros em requisições

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização e responsividade
* **JavaScript (Vanilla)** → Interatividade e requisições HTTP
* **PHP** → Backend simples (API)
* **JSON** → Comunicação entre front-end e back-end

## ⚙️ Como Executar o Projeto

1. Instale um servidor local (XAMPP, WAMP ou Laragon)
2. Coloque a pasta do projeto dentro de:

```bash
htdocs/
```

3. Inicie o Apache
4. Acesse no navegador:

```bash
http://localhost/numero-surpresa
```

---

## 💡 Como Funciona

1. O usuário digita uma pergunta na interface
2. O JavaScript envia a pergunta para o backend via **Fetch API**
3. O PHP processa a requisição
4. Uma resposta aleatória é selecionada
5. O backend retorna um JSON com a resposta
6. O front-end exibe o resultado dinamicamente na tela
